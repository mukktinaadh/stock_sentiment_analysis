# Stock Market Sentiment Analysis

## Overview
Real-time stock sentiment analysis using news headlines and social media data with VADER sentiment scoring and visualization.

## Features
- Fetch real-time financial news
- Sentiment scoring using VADER
- Price correlation analysis
- Interactive visualizations
- Export results to CSV

## Requirements
Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```

## Compatible with
- macOS M3/M4 (ARM architecture)
- Python 3.8+
